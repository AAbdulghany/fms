## Overview

Phase 3.1 completes deferred MVP items and delivers three strategic pillars: **actionable notifications**, **full invoice lifecycle**, and **assets as the sales differentiator** (scheduling, smart labels, client maintenance view). Also includes CSV import, demo hosting, and manual subscription gating.

# Phase 4 Outline (Planning Only)

**Status:** Draft — not in Phase 3.1 build scope

---

## Themes

### 1. Monetization & growth
- Stripe or regional payment gateway integration
- Self-serve tenant signup + trial automation
- Usage-based billing hooks (sites, users, assets)

### 2. Operations excellence
- Dedicated WO Requests dashboard (Option A from Phase 3 AgDR)
- SLA automation with escalation notifications
- Advanced analytics for client owners (MTBF, cost per asset)

### 3. Field & mobile
- PWA / native mobile for technicians (offline reports, photo capture)
- NFC asset tags (complement QR)
- External label printer SDK (Zebra)

### 4. Internationalization
- Multi-language invoice PDFs (AR/EN)
- RTL invoice layout polish
- Locale-aware date/currency formatting in PDFs

### 5. Platform
- Multi-region deployment
- SSO (SAML/OIDC) for enterprise tenants
- Webhook integrations (ERP, accounting)

---

```
P4-M1 Payments / signup
P4-M2 Requests dashboard + SLA
P4-M3 Mobile PWA
P4-M4 i18n invoices
P4-M5 Enterprise SSO
```
